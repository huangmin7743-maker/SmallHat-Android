🎩 SmallHat 原生安卓APP - 编译指南
=========================================

📱 跟Telegram一样的原生安卓APP！
   底部Tab导航 + 聊天列表 + 通讯录 + 工具 + 个人中心
   深色主题，Material Design 3

🚀 编译方法（需要一台电脑）：

方法1：Android Studio（推荐）
1. 安装 Android Studio
2. 打开此文件夹
3. 点菜单 Build → Build Bundle(s) / APK(s) → Build APK(s)
4. 等1-2分钟
5. APK文件在 app/build/outputs/apk/debug/

方法2：命令行（需要配置ANDROID_HOME）
1. 下载 Android SDK Command Line Tools
2. 运行 ./gradlew assembleDebug
3. APK文件在 app/build/outputs/apk/debug/app-debug.apk

📦 安装：
将 APK 传到手机 → 打开安装即可

💡 需要工具：
- Android Studio: https://developer.android.com/studio
- 或者 Gradle + Android SDK Command Line Tools
